# Spanish Grammar Curriculum (Castellano) — CEFR A1-C2

Reference framework for Celestia AI tutor. Based on the Instituto Cervantes Plan Curricular, DELE exam requirements, and CEFR descriptors. All content targets **Spanish from Spain (Castellano)** with Peninsular-specific features marked explicitly.

## How to Use This Document

- **Assessment engine**: use topic lists to identify what a student knows and where the gaps are.
- **Placement**: match demonstrated knowledge against level boundaries to assign an initial CEFR level.
- **Program generation**: build personalized modules by selecting topics the student has not yet mastered at their current level.
- **Progression logic**: topics within each level are ordered by pedagogical dependency — earlier topics are prerequisites for later ones.

## What Distinguishes Each Level

| Transition | Key Grammar Differentiators |
|---|---|
| A1 > A2 | Past tenses (indefinido, imperfecto), future, conditional, combined pronouns, basic subjunctive introduction |
| A2 > B1 | Systematic subjunctive (present + imperfect), conditional type 2, pluperfect, reported speech, verbal periphrases |
| B1 > B2 | Complete subjunctive system (4 tenses), conditional type 3, advanced relative clauses (cuyo, el cual), nuanced mood selection |
| B2 > C1 | Futuro de subjuntivo (recognition), advanced emphatic/topicalization structures, complete se system, register variation, discourse markers |
| C1 > C2 | Stylistic grammar, absolute constructions, native-level word order manipulation, complete dialectal awareness, colloquial grammar, error recognition (queísmo/dequeísmo) |
